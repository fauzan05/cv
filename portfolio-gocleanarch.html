Project Overview

Golang Clean Architecture SaaS Template adalah starter kit production-ready untuk membangun aplikasi SaaS dengan Go, mengikuti prinsip Clean Architecture. Template ini menghilangkan kerja boilerplate selama berminggu-minggu dengan menyediakan fitur SaaS esensial secara out-of-the-box: JWT Authentication (access & refresh token), Multi-Tenancy dengan organization-first design, Subscription Management dengan tiered plans, Email Verification async, serta UUID Primary Keys dan Soft Delete untuk database design yang aman dan scalable.

Clean Architecture Layers

Delivery Layer (HTTP Controllers)
auth_controller, user_controller, organization_controller, subscription_controller
↓
Usecase Layer (Business Logic)
JWT Generation, Email Verification, Role Management, Subscription Logic
↓
Repository Layer (Data Access)
GORM Operations, Query Building, Soft Delete Scoping, Transaction Management
↓
Entity Layer (Domain Models)
User, Organization, Subscription, Plan - UUID Primary Keys & Soft Delete

Database Schema

// Core Tables with UUID & Soft Delete
organizations (id, name, slug, logo_url)
users (id, org_id, email, password, verified)
organization_members (id, org_id, user_id, role)
plans (id, name, price, limits, features)
subscriptions (id, org_id, plan_id, status)
audit_logs (id, action, resource, old/new_value)
