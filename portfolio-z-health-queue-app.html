Healthcare Web Application

Z Health Queue

Smart Healthcare Queue Management System - Real-time Queue Tracking via QR Code Scan

Project Overview

Z Health Queue adalah sistem manajemen antrian kesehatan berbasis web yang memungkinkan pengunjung untuk memantau status antrian mereka secara real-time melalui smartphone. Dengan hanya melakukan scan QR Code pada struk antrian, pengunjung dapat melihat posisi antrian, estimasi waktu tunggu, dan notifikasi ketika giliran mereka tiba. Sistem ini dirancang untuk meningkatkan efisiensi pelayanan kesehatan dan memberikan kenyamanan bagi pasien tanpa harus menunggu di tempat.
