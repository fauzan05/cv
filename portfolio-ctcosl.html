Coiled Tubing Management System - Admin Dashboard for Oil & Gas Operations

Project Overview

CT-COSL (COSL Coiled Tubing Management System) adalah admin dashboard berbasis web profesional yang dibangun dengan Laravel 12 dan Vue.js 3, dirancang khusus untuk mengelola operasi coiled tubing di industri oil & gas. Aplikasi ini menyediakan tools komprehensif untuk inventory management, job tracking, dokumentasi teknis, reporting, dan user administration. Project ini melayani operasi COSL (China Oilfield Services Limited), menawarkan platform terintegrasi untuk mengelola konfigurasi toolstring, wellstack assemblies, thread specifications, dan dokumentasi operasional.
