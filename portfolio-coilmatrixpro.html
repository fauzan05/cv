Desktop Application

Coil Matrix Pro

Advanced Data Acquisition System - Real-time Sensor Monitoring for Coiled Tubing Operations

Project Overview

Coil Matrix Pro adalah aplikasi desktop profesional yang dibangun dengan WinUI 3, dirancang khusus untuk operasi coiled tubing di industri oil & gas. Aplikasi ini menyediakan solusi komprehensif untuk monitoring sensor real-time, recording data, visualisasi chart (Gauge & Line), serta export laporan dalam format Excel dan PDF. Sistem terintegrasi dengan hardware Opto 22 SNAP UP1 ADS untuk akuisisi data sensor dengan berbagai tipe sinyal (Digital Encoder ABC, Analog Standard, Digital Magnetic Pickup) dan dilengkapi dengan sistem lisensi berbasis Hardware ID untuk keamanan maksimal.

Key Features

Real-time Monitoring

Polling sensor via Opto 22 SNAP
